# Climate Risk Prediction using Ridge Regression

## Project Overview

This project was completed for the DATA SCIENCE PROJECT LIFECYCLE module.

The objective is to help the fictional reinsurance company **Aethelgard Re** assess climate risk by analysing historical greenhouse gas data and predicting future global temperature anomalies using Ridge Regression.

---

## Project Objectives

- Integrate multiple climate datasets
- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Analyse correlations between climate variables
- Detect multicollinearity using VIF
- Build a Ridge Regression model
- Evaluate model performance
- Forecast global temperature anomalies for 2022–2031
- Provide business recommendations
